# Battle-City
An Remake of Battle City in C++ / WinApi 

## Controls

- PLAY

  - z : Shoot

  - ↑,↓,←, → : Move

- Construction Mode

  - z : Tile Change
  - Enter : Map Save

## Requirements

msimg32.lib
